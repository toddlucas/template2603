# Repo Setup

## Renaming project placeholders

After cloning the repo initially, replace the following with the product name:

* Product Name - User facing names
* ProductName  - Object names
* product-name - Scripts and config files
* product_name - Database names and objects
* productrepo  - The project repo name
* product.tld  - The project web site
